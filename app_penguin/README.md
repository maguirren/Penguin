# Penguin

Generado de sitios estaticos escrito en Python.

## Instalacion

Modo editable(para desarrollo):

```bash
git clone https://github.com/maguirren/penguin.git
cd penguin
pip install -e .
```

## Usage

Crea un nuevo sitio
```bash
penguin new site MiBlog
```

Crea un nuevo post
```bash
penguin new post "Mi Primer post"
```

Construye el sitio
```bash
penguin build
```

Deploya el sitio
```bash
penguin deploy
```
